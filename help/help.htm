Introduction

vtbgen is a utility application made using basic Front-end web technologies. "vtbgen" is abbreviation for Verilog Testbench geenrator.
The motivation behind this app is that to test a design, written in Verilog, you have to define your digital signals which will be inputs.
Now, you know your inputs, you know when your input signal will be LOW, HIGH, X or Z! So instead of writing this information by hand using some text based language, why not use some graphical means to do the same?
This is exactly what this application tries to acheve.
This application provides you a graphical interface which lets you to draw as many signal as required and then with one click, it generates verilog code which can be placed in initial block of test bench.

Getting Started

This Section Describes how to get started with this application.
when you open this application, first you will see Fig1.
Lets add a new signal with a name clock. TO do this by following the steps under:

  • Goto File
  • Select New Signal from the drop down menu
  • A pop-up will be displayed where the name of signal (it should be unique) will be entered.
Fig2. show the result.

Fig2. Clock Signal added


Now, I want to change the simulation time. There will be obvious reasons for you to do that!
Fig3. shows how to change simulation time.

Fig3. Drop Down menu where there is an option to change Simulation time

To Change simulation time

  • Goto Options menu
  • Select Change Simulation Time
  • A pop up be displayed, where you will enter the required time
By Default, it is 100

*Note that, the units of the time can not be selected from here, you will have to use `timescale compiler directive in the final verilog testbench to set the units.

Fig4 shows the same signal as shown in Fig2 but with smaller simultion time (of 20).

Fig4. Simlation time of 20

You can see that, changing the simulation time, changed the simulation time for all the signals

As the name of "clock" signal suggest, it is a clock signal, so we need to change this to a clock signal. You can do that by

  • Clicking on the square box on the right side of text "Clock". This selects this signal
  • Goto Edit Drop Down Menu
  • Select convert to clock signal
  • A Pop-up will be displayed, Enter the half time period of the clock signal

Fig5. Clock Signal

Notice at the top of the signal line, you can see 1 and 0 written, these are the value of the signal at that particular instant of time.
